Exemplo 13 - CSS bootstrap

O bootstrapé um framework front-end popular para desenvolvimento de aplicações web. Ele foi criado pelo antigo Twitter com a finalidade de facilitar a criação de páginas. Ele possui um conjunto de seletores de classe que podem ser aplicada as mais diversas situaçoes de estilização de documentos.
