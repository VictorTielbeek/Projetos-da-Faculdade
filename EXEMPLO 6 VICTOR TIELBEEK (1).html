Web Semantica

O termo web semantica se refere a um conjunto de conceitos que estão interligados e que foi inicialmente cunhado pelo consorcio W3C. Essencialmente, ele diz respeito as diretrizer para tornar os armazenamentos de documentos HTML mais interoperaveis.

A web semantica é um movimento colaborativo para organizar a informação de forma mais legivel para a maquina, por meio padroes de formatação de dados. Por exemplo
