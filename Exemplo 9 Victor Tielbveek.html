Exemplo 9 - CSS

O CSS é utilizado na estilização de elementos HMTL para que você possa aplicar a folha de estilo em cascata - css nos elementos HTML você necessariamente podera utilizar as regras css que podem ser definidas por meio de seletores seguidos de chaves {} e dentro dessas chaves, propriedades e valores

Os seletores contidos nas regras css, são padrões usados para definir quais elementos HTML receberão determinados estilos. Eles podem ser de três tipos: Seletores de Tipo, ID e CLasse

Seletores de Tipo

Os seletores de tipo são aqueles que são representados pelas tags HTML, geralmente são aplicados a todas as TAGS HTML que se referenciam.

Esses seçetores são descritos da seguinte forma: h1 {propriedade:valor;}, no caso do exemplo citado o seleotr de tipo é o h1 que representa uma tag de seção.

Seletores de ID

Os seletores de ID são aqueles que utilizam IDs (atributos) únicosde elementos HTML. Esses seletores são personalissimos, pois são aplicados unicamente aos elememntos HTML que tiverem o ID específico.

Para criar um seletor de ID, seguimos a senguinte sintaxe: #NomeSeletor{propriedade:valor;}

Seletores de Classes

Os seletores de classes são aqueles usados para aplicar o estilo a varios elementos HTML que possuem o mesmo atributo class. Sua sintaxe é definida da seguinte maneia: .NomeSeletor{propriedade:valor}
