Exemplo 11 - CSS Formas de vinculação

Existem três formas básicas de vinculação de arquivos CSS. A vínculação externa, a inline e a incorporada.

Neste exemplo usaremos a vinculação externa, que é aquela onde as regras CSS são definidas no documento externo html.

Esse tipo de vinculação geralmente deve ser utilizada na maior parte das vezes. Sua grande vantagem é que ele pode ser reaproveitado em vário documentos html, facilitando a sua manutenção. quando tivermos mais páginas.

Vinculação Externa

A vinculação externa é a forma mais comum e recomendada de aplicar estilos. Podemos utilizar essa vinculação de várias formas, ou criando um arquivo CSS externo ao projeto ou simplesmente utilizando um arquivo CSS disponível ns internet, como por exemplo na utilização do Bootstrap
